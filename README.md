# Laravel Design Copy Button

Copy to clipboard button component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/copy-button
```

## Usage

### Livewire Component

```blade
<livewire:ld-copy-button />
```

### Blade Component

```blade
<x-ld-copy-button />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-copy-button-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-copy-button-views
```
